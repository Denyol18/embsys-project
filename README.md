# Beágyazott rendszerek gyakorlat projektmunka

Megvalósított projekt: Froggeren alapuló játék

## Játék kipróbálása:

1. simavr beüzemelése a következő alapján: https://gist.github.com/akosthekiss/e8c1c1775d66dba8de1c7946aa6b973e
2. Az ```atmega128_minifrog.c``` fájl elhelyezése a ```simavr/examples/board_olimex_avr_mt128/``` mappába
3. ```make``` parancs futtatása
4. ```./obj-x86_64-linux-gnu/mt128.elf ./atmega128_minifrog.axf``` parancs futtatása
5. Jó játékot! :)
